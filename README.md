## Features

# Build Android:
- cd ./android
+ Generate APK:
- ./gradlew assembleRelease
+ Generate Release Bundle (file .aab):
- ./gradlew bundleRelease

+ Quick command
- cd ./android && ./gradlew assembleRelease && cd ..
- cd ./android && ./gradlew bundleRelease && cd ..
- react-native run-android --variant=release

+ check task:
./gradlew tasks
